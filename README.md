# Portfolio Web - Abel Arias

Portfolio web construido con Angular 19, Three.js y GSAP.

## 🚀 Tecnologías

- Angular 19
- TypeScript
- Three.js
- GSAP
- Framer Motion
- Lottie Web

## 🛠️ Instalación

```bash
# Clonar repositorio
git clone https://github.com/atomicplus99/portfolio-me.git

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm start
```
